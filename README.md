# gherkin-rust

A pure Rust implementation of the Gherkin (`.feature` file) language for the Cucumber testing framework.

There is a native [cucumber-rust](https://github.com/bbqsrc/cucumber-rust) test running using this crate.

## Usage

```toml
[dependencies]
gherkin-rust = "^0.1"
```
